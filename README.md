Lung Detect AI

An AI-powered web application that analyzes chest X-ray images to detect and classify common lung diseases using deep learning.
🔗 Live Demo: https://replit.com/@dhilangurtoo/Lung-Detect
What it Does

Lung Detect uses a convolutional neural network (CNN) trained on chest X-ray images to identify signs of COVID-19, pneumonia, tuberculosis (TB), and lung opacity, while also recognizing healthy (normal) lungs. Users can upload a chest X-ray through the web interface, where the image is automatically preprocessed before being analyzed by the model. Within seconds, the application returns the predicted condition and confidence score, providing a fast demonstration of how computer vision and deep learning can assist in the early screening of respiratory diseases. The project is intended for educational and research purposes and is not a substitute for professional medical diagnosis.
